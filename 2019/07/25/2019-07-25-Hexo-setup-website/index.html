Hexo + GitHub 建站指北

Hexo 是什么？

Hexo 是一个开源、简单、轻量级，而且很强大的博客框架，主要使用 Nodejs 实现；非常适合喜欢使用 Markdown 文档来写博客的伙伴；它通过将 Markdown 文件转换为静态网页，来实现 Web 访问。它还支持很多可以定制功能，如:插件（Plugin）和主题（Themes），可以根据自己的喜好进行个性化设置。这里是他的文档

Hexo 的建站步骤

  • 安装 Git
  • 安装 NodeJs
  • Github 创建项目

需要注意的地方

由于在 Github 上创建项目仓库时，默认访问路径为 ”http(s)://.github.io/“ 所以在设置根目录下的配置文件 ”_config.ymal“ 中的 url 配置项和 root 配置项时，需要将 url 设置为 ”http(s)://.github.io/“，并把 root 设置为 *”//“，而非默认的 *”/“
